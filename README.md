## ğŸ® How to Play

### Objective
Survive as long as possible by avoiding obstacles. The longer you run, the higher your score!

### Controls

**Desktop:**
- **Spacebar** - Jump
- **Arrow Down** - Duck (when running)
- **Spacebar (Game Over)** - Restart

**Mobile:**
- **Tap Screen** - Jump
- **Swipe Down** - Duck
- **Tap (Game Over)** - Restart

### Game Mechanics

- **Speed increases** - Game gets progressively faster
- **Cacti obstacles** - Jump over them
- **Pterodactyls** - Duck under flying dinosaurs
- **Score** - Based on distance traveled
- **Day/Night** - Theme changes every 700 points
- **Collision** - Hit any obstacle = game over

### Scoring
- **1 point** = Every frame survived
- **Approximately 100 points** = 1 second of survival
- **High score** is displayed at top right
- Challenge yourself to beat your record!
